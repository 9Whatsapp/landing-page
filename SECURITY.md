# Security Policy

| Versi | Didukung |
|-------|----------|

## Melaporkan Kerentanan

Jika Anda menemukan kerentanan keamanan, jangan buat issue publik. Kirim laporan melalui:

- **Email**: security@9whatsapp.web.id

Mohon sertakan:

- Deskripsi kerentanan
- Langkah reproduksi
- Dampak yang mungkin terjadi
- Saran perbaikan (jika ada)

## Respons

- Konfirmasi penerimaan dalam 3 hari kerja
- Estimasi perbaikan dalam 7 hari kerja
- Update berkala hingga resolved

## Scope

- Source code di repository ini
- Tidak termasuk dependensi pihak ketiga (laporkan ke maintainer masing-masing)

## Tidak Termasuk

- Vulnerability di dependensi eksternal
- Issues yang membutuhkan akses fisik
- Social engineering
- Spam atau serangan DDoS
